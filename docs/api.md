# API Draft

## Sensor

- `GET /api/sensor/latest`
- `GET /api/sensor/summary`
- `GET /api/sensor/trend?mode=realtime|hour|day|week&start=2025-11-01T00:00:00&end=2025-11-30T23:59:59&limit=300`

## Point Cloud

- `GET /api/point-cloud/list?start=2025-11-01T00:00:00&end=2025-11-30T23:59:59`
- `GET /api/point-cloud/latest`
- `GET /api/point-cloud/file/{fileName}`
